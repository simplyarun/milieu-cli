    ```typescript
    interface OpenApiResult {
      check: Check;
      detected: boolean;  // for ctx.shared.openApiDetected (Bridge 3)
    }
    export async function checkOpenApi(baseUrl: string, timeout?: number): Promise<OpenApiResult>;
    ```

    Implementation details:

    1. Define OPENAPI_PATHS constant array (9 paths):
       `/openapi.json`, `/swagger.json`, `/api-docs`, `/v3/api-docs`, `/v2/api-docs`, `/swagger/v1/swagger.json`, `/api/openapi.json`, `/api/swagger.json`, `/.well-known/openapi.json`

    2. Fire all 9 httpGet calls via Promise.all with Accept header `"application/json, text/plain, */*"`

    3. For each successful (ok: true) response, validate:
       a. Content-Type is NOT text/html or application/xhtml+xml (reject Swagger UI pages)
       b. For JSON types (application/json, application/vnd.oai.openapi+json): JSON.parse body, check for top-level "openapi" or "swagger" key
       c. For YAML types (application/yaml, text/yaml, application/x-yaml, application/vnd.oai.openapi): test body with `/^(openapi|swagger):/m` regex
       d. For unknown Content-Type: if body starts with "{", try JSON.parse fallback

    4. Extract info from first valid hit:
       - JSON: version from parsed[specType], endpointCount from Object.keys(parsed.paths ?? {}).length
       - YAML: version via regex `/^(openapi|swagger):\s*["']?(\d+\.\d+(?:\.\d+)?)/m`, endpointCount via `/^  \/\S+:/gm` match count

    5. Return Check with:
       - Pass (JSON): `{ id: "openapi_spec", label: "OpenAPI Spec", status: "pass", detail: "OpenAPI {version} found with {N} endpoints", data: { version, specType, endpointCount, path } }`
       - Pass (YAML with version): `{ status: "partial", detail: "OpenAPI {version} found (YAML format)", data: { version, specType, endpointCount, path } }`
       - Fail: `{ status: "fail", detail: "No OpenAPI spec found" }`

    6. Return `detected: true` if any valid spec found (pass or partial), false otherwise.
