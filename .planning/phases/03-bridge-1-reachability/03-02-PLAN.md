**`src/bridges/reachability/meta-robots.ts`:**

```typescript
import type { Check } from "../../core/types.js";

export function checkMetaRobots(html: string): Check
export function checkXRobotsTag(headers: Record<string, string>): Check
```

**checkMetaRobots implementation:**
1. Extract head content: match `<head[^>]*>([\s\S]*?)<\/head>` (case-insensitive flag). If no match, return pass.
2. Within head content, find all meta tags matching robots-related names. Use regex:
   - Pattern A (name first): `<meta\s+[^>]*name\s*=\s*["'](robots|googlebot|bingbot)["'][^>]*content\s*=\s*["']([^"']*)["'][^>]*\/?>`
   - Pattern B (content first): `<meta\s+[^>]*content\s*=\s*["']([^"']*)["'][^>]*name\s*=\s*["'](robots|googlebot|bingbot)["'][^>]*\/?>`
   - All regex flags: `gi`
3. Collect all content values from matches. Split each on `,` and trim.
4. If any directive is "noindex" → status "fail", detail lists found directives
5. If any directive is "nofollow" (but no noindex) → status "partial", detail lists found directives
6. Otherwise → status "pass"
7. Check id: `"meta_robots"`, label: `"Meta Robots Tags"`
8. Store found directives in `data: { directives: string[] }` (empty array if none found)

**checkXRobotsTag implementation:**
1. Read `headers["x-robots-tag"]` (headers object has lowercase keys from Phase 2 httpGet)
2. If not present → status "pass", detail "No X-Robots-Tag header"
3. Split value on `,`, trim each directive
4. If any directive contains "noindex" → status "fail"
5. If any directive contains "nofollow" or "noarchive" or "none" → status "partial"
6. Otherwise → status "pass" (header present but no restrictive directives)
7. Check id: `"x_robots_tag"`, label: `"X-Robots-Tag Header"`
8. Store in `data: { directives: string[], raw: headerValue }`

Tests for checkXRobotsTag:
- No x-robots-tag header → "pass"
- x-robots-tag: "noindex" → "fail"
- x-robots-tag: "nofollow" → "partial"
- x-robots-tag: "noarchive" → "partial"
- x-robots-tag: "none" → "partial" (none = noindex+nofollow, but treat as partial since it's a general restriction)
- x-robots-tag: "noindex, nofollow" → "fail" (noindex wins)
- x-robots-tag: "all" → "pass"
